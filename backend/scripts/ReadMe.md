backend/check-db.js: вспомогательный скрипт. Проверяет подключение к БД, перечисляет пользователей и при отсутствии — создает admin. Полезен вручную при отладке/первичной проверке.
backend/migrate-db.js: утилита для sequelize.sync({ alter: true }) и (при необходимости) создания admin. Применяется разово при обновлении схемы.
backend/migrate-role.js: точечная миграция MySQL — добавляет колонку role в таблицу authentications. Нужна один раз для изменения схемы.
